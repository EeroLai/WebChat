# WebChat-Nuxt3-SignalR

Simple WebChat through Fornt-End Nuxt and Back-End .Net Core develop

## Front-End
### use
- TailWindCSS
- micosoft/SignalR
- pinia/nuxt (npm install pinia @pinia/nuxt --legacy-peer-deps) RC有相容性問題

## Back-End
### use
- CORS
- SignalR
- Swagger

### run server

```
npm install
```

```
npm run dev -o
```
